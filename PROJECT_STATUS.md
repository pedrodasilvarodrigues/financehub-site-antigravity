# Projeto
FinanceHub Site Antigravity

# Objetivo
Site institucional do FinanceHub em React + Vite, extraido do pacote `financehub-site.zip` gerado pelo Antigravity/Gemini.

# Concluido
- Projeto extraido para uma pasta propria em `Documents/Codex/financehub-site-antigravity`.
- Estrutura React + Vite preservada com paginas institucionais, componentes, assets, estilos, README e configuracao de build.
- Dependencias instaladas com `npm ci` e build de producao validado com `npm run build`.
- Repositorio GitHub criado e publicado em `https://github.com/pedrodasilvarodrigues/financehub-site-antigravity`.
- Release `v1.0.0` criada no GitHub com o instalador `FinanceHub-Setup-1.0.0.exe`.
- Pagina Downloads atualizada para baixar o instalador real pelo botao "Baixar FinanceHub para Windows".
- Site publicado na Vercel em `https://financehub-site-antigravity.vercel.app`.
- Textos institucionais revisados para remover promessas que nao existem na versao atual e destacar apenas funcionalidades reais do FinanceHub.
- Aba Contato recebeu card de treinamento com botao para WhatsApp no numero (32) 98709-9260 e mensagem pronta para empresas solicitarem treinamento da equipe.
- Site institucional ajustado para linguagem de produto real: removidos links de GitHub/LinkedIn, página de planos, menções técnicas a Supabase, TCC, repositório, conteúdo acadêmico e textos demonstrativos das páginas visíveis.
